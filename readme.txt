Skip easily to next/previous when editing posts in the admin

== Installation ==

You can download and install Edit Next Post using the built in WordPress plugin installer. If you download Edit Next Post manually, make sure it is uploaded to "/wp-content/plugins/edit-next-post/".

Activate Edit Next Post in the "Plugins" admin panel using the "Activate" link. If you're using WordPress Multisite, you can choose to activate Edit Next Post network wide for full integration with all of your sites.
